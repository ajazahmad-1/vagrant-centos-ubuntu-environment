# DevOps Vagrant Virtual Machines Lab

This repository contains a hands-on DevOps lab where I practiced creating and managing virtual machines using Vagrant.

## Tools Used

- Vagrant
- VirtualBox
- Git Bash
- CentOS Stream 9
- Ubuntu Jammy 22.04

## Project Overview

In this lab, I created two separate virtual machine environments:

1. CentOS Stream 9 VM
2. Ubuntu Jammy 22.04 VM

Using Vagrant, I learned how to:

- Initialize a Vagrant environment
- Download Vagrant boxes
- Start and stop virtual machines
- Connect to VMs using SSH
- Manage multiple Vagrant environments
